Infrastructure Cost & Efficiency Analyzer

Identify waste, optimize executor packing, and get a shareable PDF report for Spark workloads.

Both paths produce the same full ICEA report and payment flow. Enter data manually (Path 1) or upload a Spark event log (Path 2); then review the analysis, choose a tier, and get your report.

Overview

Two ways to analyze

Manual entry (Path 1) or event log upload (Path 2). Either way you get a preview, tier choice, and the full ICEA report after payment.

  • Path 1 — Manual entry: Configure cluster and workload in the form → see live preview → choose tier → pay and get report (or request expert).
  • Path 2 — From event log: Upload a Spark event log (.json, .json.gz, or .zip) → we derive workload from the log → same preview, tier choice, and full report after payment.
